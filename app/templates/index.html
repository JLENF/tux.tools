Current tools:

$ geo.tux.tools - get information for ip

How to use:

  • curl https://geo.tux.tools/8.8.8.8 in terminal to get all ip information (json format)
  • curl https://geo.tux.tools/man to show man page

----------------------------------

$ bin.tux.tools - command line pastebin for sharing terminal output.

How to use:

  • echo just testing! | nc bin.tux.tools 9999
  • cat file.txt | nc bin.tux.tools 9999

the temporary files (links) are deleted in 24h!

----------------------------------

$ tux.tools/password - password generator for terminal.

How to use:

  • curl tux.tools/password - generate password with 12 characters by default
  • curl tux.tools/password/20 - generate password with another length, eg. 20 characters.
